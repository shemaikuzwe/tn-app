## TN APP

****
create TN app is the best way to create Next js app

-  Initialize ORM
   * Drizzle
   * Prisma
- Initialize Database

     * Postgres(local)
     * Neon Postgres
     * Vercel Postgres
-  Initialize  shadcn
-  Initiaze Auth js


***

### How to use

1. Install
 ```bash
pnpm i -g tn-app
```
2. Create TN App
```
pnpm create-tn-app
```
